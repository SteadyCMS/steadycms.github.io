# SteadyCMS website

The official SteadyCMS website at [steadycms.com](https://steadycms.com).


## Development

```bash
hugo server

```

To re-build the CSS during development, open another terminal and run the following:

```bash
cd themes/steadycms
```

```bash
npm run watch
```


## Deployment

Build CSS for production:

```bash
npm run build
```
