# SteadyCMS website

The official SteadyCMS website at [steadycms.com](https://steadycms.github.io).


## Development

```bash
hugo server

```

To re-build the CSS during development, open another terminal and run the following:

```bash
cd themes/steadycms
```

```bash
npm run watch
```


## Deployment

Minify CSS for production:

```bash
npm run minify
```
